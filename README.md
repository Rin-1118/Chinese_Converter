# Chinese_Converter

スマホ優先の中国語変換アプリです。ブラウザ上で次の2系統を扱います。

- 日本語の文や単語を入力して、中国語訳・簡体字・拼音を表示
- 中国語文を入力して、選択した言語の意味と拼音へ変換
- 中国語や翻訳結果をブラウザの音声機能で発音

## 重要な制約

日本語から中国語、中国語から選択言語への翻訳は、外部の MyMemory Translation API を使います。Google公式翻訳APIではありません。

外部APIを使うため、インターネット接続が必要です。また、MyMemory API の仕様に合わせて、1回の日本語入力は500バイト以内にしています。長い文章は短く分けて入力してください。

中国語から拼音への変換は、ブラウザ内で処理します。

発音はブラウザの音声合成機能を使います。端末やブラウザに対象言語の音声が入っていない場合、読み上げ品質や対応状況が変わります。

## 使い方

1. `日本語` タブに日本語の文や単語を入力します。
2. `翻訳` を押すか、`Shift + Enter` で中国語訳・簡体字・拼音が表示されます。
3. `Languages` ページで、中国語入力の翻訳先を選びます。
4. `中国語` タブに繁体字・簡体字の中国語文を入力します。
5. `変換` を押すか、`Shift + Enter` で上に選択した言語の意味、下に拼音が表示されます。
6. スピーカーボタンを押すと、表示された中国語や意味を発音します。
7. 必要なら `履歴` を押して、最近の翻訳・変換を保存します。
8. `History` ページで保存済みの履歴を確認・復元できます。
9. 右上のテーマボタンでライトモードとダークモードを切り替えます。

履歴はブラウザの `localStorage` に保存されます。別ブラウザや別端末には自動同期されません。

## 依存関係

- MyMemory Translation API: 日本語から中国語、中国語から選択言語への翻訳に使用。
- OpenCC JS `opencc-js` 1.3.1: 簡体字・繁体字・日本漢字系の変換に使用。
- pinyin-pro `pinyin-pro` 3.28.1: 漢字から拼音への変換に使用。
- Vite `vite` 8.0.16: ローカル開発・ビルドに使用。
- Lucide `lucide` 1.21.0: UIアイコンに使用。
- Web Speech API: ブラウザ内の音声合成に使用。

依存関係のバージョンは `package.json` を確認してください。

## 実装上の注意

- 日本語翻訳は MyMemory API の翻訳結果を受け取り、その中国語テキストを簡体字化してから拼音を生成します。
- 中国語変換は内部で `opencc-js` の `jp -> cn`、`tw -> cn`、`hk -> cn` 変換を順に通します。
- 拼音と発音は簡体字化後の文字列から生成します。
- ユーザー辞書登録、内蔵辞書検索、AIによる訳語補完は行いません。
